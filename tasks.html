Assigning real projects: Employers can provide tasks that reflect real-world problems, helping students practice.

Tasks

Task 1

Create a simple, clean, and responsive Profile Card using only HTML and CSS.

The card should be centered on the page.
The card width should be between 300–400p Add padding and rounded corners.
Inside the card, include:
A circular profile image.
Your name (as a heading).
A short bio (2–3 sentences).
A row of social media icons or placeholder links.

Task 2

Write a C++ program that:

Reverses a String.
Write a Python function that takes a string as input and returns the string reversed.
Challenge yourself:
Modify it to ignore spaces when reversing.
Or make it work for full sentences while keeping word order intact.
Hint: Define a new function that can reverse the word
Validate that the only accepted input is text
If invalid, show an error

Task 3

Write a C++ program that:

Create a simple command-line To-Do List application where the user can:
Add tasks
View all tasks
Remove a task by number
Exit the program
Use a list to store tasks.
Show a menu inside a while loop.
Use input() for user choices.
Validate invalid selections.
Optional: Save tasks to a text file.
